# PolyJournal

Diário de trading pessoal para Polymarket. PWA que funciona como app no celular.

## Features
- Registro de trades com stake, odd e resultado
- Tag planejada vs impulsiva
- Nível de confiança (1-5 estrelas)
- Bankroll tracker com limite de risco
- Dashboard com PnL, win rate, ROI, streak
- Insights: confiança vs resultado, performance por dia da semana
- PWA: instala como app no celular
